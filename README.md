# chesster
AI Chess Robot Project  

## Installation  
Requirements: `pipenv`.  
If `pipenv` not installed but `pip` is:
```bash
pip install pipenv
```
Finally, install the dependencies with
```bash
pipenv install
```  
and 
```bash
pipenv shell
```

## Run  
After following the Installation steps
```bash
pipenv run main
```
